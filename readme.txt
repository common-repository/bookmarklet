== Description ==

The Bookmarklet WordPress plug-in gives you the ability to add social media sharing options to the Posts and Pages of your website.

Currently supports sharing for 22 of some of the most popular social media sites including and Addthis for more bookmark/share options.:

* Blinklist
* Bebo
* Delicious
* Designfloat
* Diigo
* Digg
* Email
* Facebook
* Friendfeed
* Google Bookmarks
* Google plus
* Linkedin
* Myspace
* Newsvine
* Orkut
* Pinterest
* Reddit
* Stumbleupon
* Technorati
* Tumblr
* Twitter
* Yahoo Buzz


== Installation ==

1. Upload  folder `bookmarklet` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set settings in dasboard
1. Enjoy it. :)
